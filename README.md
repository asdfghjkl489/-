Руководство по использованию переводчика Морзе
Запустите программу на вашем устройстве и дождитесь полной загрузки интерфейса. Далее найдите на экране переключатель языка — обычно он расположен в верхней части интерфейса — и выберите нужный язык: «Русский» для работы с русской азбукой Морзе либо «Английский» для работы с английской азбукой Морзе.
Чтобы ознакомиться с таблицей азбуки Морзе для выбранного языка, нажмите кнопку «Показать таблицу» или «Пример азбуки». В открывшемся окне отобразится соответствие букв и символов коду Морзе. После ознакомления закройте таблицу — для этого нажмите кнопку «Закрыть» либо кликните за пределами окна таблицы.
Для шифрования текста (перевода обычного текста в азбуку Морзе) перейдите в режим шифрования, если в программе предусмотрены отдельные режимы работы. Введите текст в поле «Исходный текст» или «Введите текст», убедившись, что он соответствует выбранному языку и не содержит неподдерживаемых символов (например, специальных знаков или цифр — они могут быть проигнорированы либо заменены). Затем нажмите кнопку «Зашифровать» или «Перевести в Морзе». Результат появится в поле «Результат» или «Зашифрованный текст»: буквы будут отделены пробелом, а слова — двойным пробелом либо специальным разделителем.
Для дешифрования текста (перевода кода Морзе в обычный текст) перейдите в соответствующий режим, если он предусмотрен. Введите код Морзе в поле «Код Морзе» или «Введите азбуку Морзе», используя точки (.) для коротких сигналов и тире (-) для длинных. Соблюдайте правила форматирования: отделяйте буквы одним пробелом, а слова — двойным пробелом. Нажмите кнопку «Дешифровать» или «Перевести из Морзе», после чего расшифрованный текст появится в поле «Результат» или «Расшифрованный текст».
Чтобы скопировать результат, выделите текст в поле вывода и используйте сочетание клавиш Ctrl+C либо команду «Копировать» в контекстном меню. Для сохранения результата в файл нажмите кнопку «Сохранить» или «Экспорт», выберите формат файла (например, .txt), укажите имя файла и папку для сохранения, затем подтвердите действие кнопкой «Сохранить».
Если нужно начать новую операцию, очистите поля ввода и вывода — для этого воспользуйтесь кнопкой «Очистить» или «Сбросить».
При возникновении проблем проверьте следующие моменты: если программа не распознаёт буквы, убедитесь в правильности выбранного языка и отсутствии ошибок в тексте; если в результате появляются вопросительные знаки или пропуски, проверьте исходный текст на наличие неподдерживаемых символов; если дешифрование даёт неверный результат, перепроверьте ввод кода Морзе и корректность расстановки пробелов; если таблица Морзе не отображается, попробуйте перезапустить программу или проверить настройки отображения.
В завершение рекомендуется сохранять важные результаты, чтобы избежать потери данных, а также сверяться с таблицей Морзе при работе со сложными или незнакомыми символами. При необходимости воспользуйтесь разделом «Справка» или «Помощь» внутри приложения для получения дополнительных сведений.

Guide to Using the Morse Code Translator
Launch the program on your device and wait for the interface to load completely. Next, locate the language switch on the screen — it’s usually positioned at the top of the interface — and select the desired language: “Russian” to work with the Russian Morse code alphabet or “English” to work with the English Morse code alphabet.
To view the Morse code table for the selected language, click the “Show Table” or “Morse Code Example” button. The window that opens will display the correspondence between letters and Morse code symbols. Once you’ve reviewed the table, close it by clicking the “Close” button or by clicking outside the table window.
To encrypt text (convert plain text into Morse code), switch to encryption mode if the program offers separate operation modes. Enter the text in the “Original Text” or “Enter Text” field, making sure it matches the selected language and doesn’t contain unsupported characters (such as special symbols or numbers — these may be ignored or replaced). Then click the “Encrypt” or “Convert to Morse” button. The result will appear in the “Result” or “Encrypted Text” field: letters will be separated by a single space, and words by a double space or a special delimiter.
To decrypt text (convert Morse code into plain text), switch to the appropriate mode if available. Enter the Morse code in the “Morse Code” or “Enter Morse Code” field, using dots (.) for short signals and dashes (-) for long ones. Follow the formatting rules: separate letters with a single space and words with a double space. Click the “Decrypt” or “Convert from Morse” button, and the decrypted text will appear in the “Result” or “Decrypted Text” field.
To copy the result, select the text in the output field and use the Ctrl+C keyboard shortcut or the “Copy” command in the context menu. To save the result to a file, click the “Save” or “Export” button, choose a file format (e.g., .txt), specify a file name and destination folder, then confirm the action by clicking “Save”.
If you need to start a new operation, clear the input and output fields by using the “Clear” or “Reset” button.
If you encounter any issues, check the following: If the program doesn’t recognize letters, verify that the correct language is selected and that there are no errors in the text. If question marks or gaps appear in the result, check the original text for unsupported characters. If decryption produces an incorrect result, double‑check your Morse code input and the spacing between letters and words. If the Morse code table doesn’t display, try restarting the program or checking the display settings.
Finally, it’s recommended to save important results to avoid data loss, and to consult the Morse code table when working with complex or unfamiliar symbols. If needed, use the “Help” or “Support” section inside the app to get additional information.
